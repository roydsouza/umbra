# Project Umbra: Directory Contents

This file provides an alphabetical index of the documentation and configuration files managed within Project Umbra.

### 📂 Configuration Files

- [`arti.toml`](arti.toml): The primary configuration file for the **Arti** (Rust) client. Manages SOCKS ports (`9050`), logging, and local storage paths.
